# lovedraw 계정 및 데이터 삭제 요청 안내

이 페이지는 lovedraw 계정 및 앱 데이터 삭제 요청 방법을 안내합니다.

## lovedraw 계정 및 데이터 삭제 요청

lovedraw를 사용하면서 생성된 계정 및 앱 데이터를 삭제하려면 아래 방법으로 요청할 수 있습니다.

## 삭제 요청 방법

아래 정보를 포함해 운영자 이메일로 요청해 주세요.

- 요청 유형: 계정 삭제 / 일부 데이터 삭제
- lovedraw에 로그인한 Google 이메일 주소
- 삭제를 원하는 방 이름 또는 초대 코드가 있다면 함께 기재
- 본인 확인과 처리를 위해 필요한 추가 연락 가능 이메일

운영자 이메일: `LOVEDRAW_SUPPORT_EMAIL`

## 삭제되는 데이터

계정 삭제 요청이 처리되면 lovedraw 서비스 제공을 위해 저장된 다음 데이터가 삭제될 수 있습니다.

- Firebase 사용자 ID와 Google 로그인 프로필 정보
- 참여한 방 정보와 방 멤버십
- 화면 위 낙서, 낙서 스냅샷
- 러브노트, 데이트 플랜, 버킷리스트, 퀴즈, Daily Spark 답변
- 사용자가 직접 선택해 업로드한 이미지와 추억 기록
- Distance Widget에서 직접 공유한 위치 기록

법적 의무, 보안, 부정 사용 방지, 분쟁 대응 등 정당한 사유가 있는 일부 기록은 필요한 기간 동안 보관될 수 있습니다.

## 처리 기간

삭제 요청은 접수 후 `LOVEDRAW_DELETION_PROCESSING_PERIOD` 내 처리합니다.

## 앱 안에서 요청하기

lovedraw 앱의 로그인 화면, 홈 화면, 방 설정 화면에 있는 `계정 삭제 요청` 버튼에서도 이 페이지로 이동할 수 있습니다.

## 문의

운영자: `LOVEDRAW_OPERATOR_NAME`

이메일: `LOVEDRAW_SUPPORT_EMAIL`

시행일: `LOVEDRAW_POLICY_EFFECTIVE_DATE`
